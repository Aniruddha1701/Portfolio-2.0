# **App Name**: Vizfolio

## Core Features:

- Interactive Portfolio: Interactive project showcase with filtering by category.
- Skill Visualization: Dynamic visualization of skills proficiency with tooltips.
- Code Playground: Integrated code playground for testing and demonstrating small code snippets.
- Theme Toggle: Customizable theme toggle for light and dark modes.
- AI Code Explainer: AI powered "code explainer tool" that provides explanation of your code.

## Style Guidelines:

- Primary color: Electric blue (#7DF9FF) for a vibrant and modern feel.
- Background color: Dark gray (#28282B) to provide a contrasting backdrop for the electric blue.
- Accent color: Lime green (#32CD32) for highlighting interactive elements and calls to action.
- Body and headline font: 'Inter' (sans-serif) for a clean and modern look.
- Code font: 'Source Code Pro' (monospace) for displaying code snippets.
- Use simple, outlined icons for a modern aesthetic.
- Responsive design to ensure optimal viewing experience across devices.
